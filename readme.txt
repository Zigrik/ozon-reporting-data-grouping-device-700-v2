Файл для обработки положить в ту же папку, где находится *.exe файл и файл settings.txt

После обработки добавятся две вкладки "grouping" и "grouping возвраты" в которых будут находится объединенные данные для того чтобы суммировать данные в новый столбец нужно указать исходный и итоговый столбец через разделитель # в файле settings.txt
Формат: "искодный столбец" # "итоговый столбец"

При указании после итогового столбца значения "# Д" будет добавляться соответствующая буква в столбец "Вид" при ненулевых значениях.
